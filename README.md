# FodderMap

**External Attack Surface Mapping Tool**
Built as a learning journey in Python.

## Current Status

- **Phase 1.1** - Core Setup + Basic CLI (complete)
- **Phase 1.2** - Passive DNS Enumeration (in progress)

## Goals

FodderMap is being built to map the external attack surface of orgainizations starting from minimal input (a domain or ASN). The project emphasizes:

- Deep understanding of Python and networking concepts
- Minimal dependencies in early phases (for learning)
- Clean, modular, and well-documented code
- Generating high-quality technical write-ups along the way

## Usage (current)

```bash
python foddermap.py -t example.com
```

## Philosophy

This project is intentionally being built with a "learn by building" approach. Core functionality is written from scratch where possible and practical to deepen understanding, then pragmatic decisions are made about when to use well-chosen third-party libraries.
